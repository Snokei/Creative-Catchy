## 🤝 Contributing
Found an amazing repository? Open a Pull Request!
1. Fork the repo.
2. Create a feature branch: `git checkout -b my-new-link`
3. Commit your changes: `git commit -am 'Add an awesome new repo'`
4. Push and submit a PR.
